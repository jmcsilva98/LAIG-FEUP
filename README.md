## Project 3 (TP3)

**3D Interface of a game** <br>
3D game interface developed in WebGL to represent the game states of a board game, Clobber, developed in SICStus Prolog.<br>

Implemented Features:
- [X] Illumination.
- [x] Different game scenarios.
- [x] Game Mode: Player vs Player, Player vs Bot, Bot vs Bot.
- [x] Game Difficulty: Rookie, Pro.
- [x] Switch view during game, which includes a camera animation.
- [x] Game Menu: Start Game, Quit Game, Undo, Game Movie (see the last game played - replay).
- [x] Game Score.
- [x] Game info.
- [x] Modeling and movement of game pieces.
- [x] Current player information.

## Build
Inside the project folder open terminal and insert:
```
python3 -m http.server 8080
```
Then go to:
```
http://localhost:8080/
```
